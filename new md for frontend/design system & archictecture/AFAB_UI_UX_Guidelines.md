# AFAB UI/UX Guidelines

## Principles

-   Minimal
-   Spacious
-   Premium
-   Accessible

## Animations

-   200ms transitions
-   Framer Motion
-   Fade + Slide
-   Scale on hover

## Responsive

Desktop first Tablet Mobile

## Loading

-   Skeletons
-   Progress bars
-   Empty states

## Accessibility

-   Keyboard navigation
-   Visible focus
-   WCAG AA contrast
-   RTL support

## Themes

Light and Dark with identical layouts.

## Components

Always reuse components. Never duplicate UI.
